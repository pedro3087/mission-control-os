# Un dashboard que opera tu portafolio SaaS con agentes de IA

Si manejas mas de un SaaS, conoces el dolor: 12 pestanas abiertas, saltar entre Vercel, GitHub, Stripe, Analytics. Un error en produccion y tardas minutos en saber cual proyecto fue, que lo causo, y como arreglarlo.

Mission Control resuelve eso.

---

Es un cockpit open-source con tres paneles:

**Portfolio** — Todos tus proyectos en una vista. Revenue, usuarios, uptime, errores. De un vistazo sabes quien esta sano y quien necesita atencion.

**Factory Brain** — Aprende patrones entre tus proyectos. Que design system convierte mejor, que estrategia de pricing funciona, que stack da menos problemas. Con nivel de confianza y evidencia real. Tu proximo proyecto arranca con ventaja.

**Guardian** — Monitorea produccion en tiempo real. Cuando detecta un error critico, analiza el stack trace, genera el fix y abre el PR. Tu solo apruebas el merge. Auto-fix con freno humano.

---

5 agentes de IA trabajan en paralelo, cada uno con su rol:

- **OS Agent** — orquesta todo el portafolio
- **Portfolio** — analiza revenue y crecimiento
- **Guardian** — arregla bugs y abre PRs
- **Brain** — recuerda que funciono entre proyectos
- **Acquisition** — trae trafico y usuarios

Cada agente es configurable: modelo, herramientas, permisos. Las acciones peligrosas requieren tu aprobacion. Control total sin micromanagement.

---

Open-source, AGPL-3.0. Next.js + React + Claude Agent SDK.
